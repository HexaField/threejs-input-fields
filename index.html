Input fields for Threejs

To add input-form-like fields inside threejs projects

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Input fields for Threejs.

The aim for this library is to enable de use of form input fields on the Threejs environment.

This library is being developed using:

  • Backbonejs, to get a structured code with a MVC development
  • Require.js, a javaScript file and module loader
  • Bower, to get an manage all the needed libraries to develop this one

How to install

You can download it from Github or fork it.
You can also get it on bower. 

$bower install threejsinputfields

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
